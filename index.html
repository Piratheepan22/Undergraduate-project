Overview

Welcome to my undergraduate project portfolio. This website documents my progress on Remote sensing of Environment Using Hyperspectral imaging. Here you'll find weekly updates, documentation, and resources related to my project work.

About Project

This project explores the use of hyperspectral imaging (HSI) and remote sensing for geological data extraction in Sri Lanka. It focuses on developing data-driven algorithms to analyze freely available satellite datasets for the precise identification of geological features. By integrating advanced spectral analysis techniques with cutting-edge HSI technology, the project aims to support geological mapping, sustainable resource management, and environmental monitoring across the region.
